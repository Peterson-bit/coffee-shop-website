# "Czarny Puls" - Modern Coffee Shop Website

This repository contains the source code for a modern, stylish website for a fictional premium coffee shop named "Czarny Puls". The project was built from scratch as a Single Page Application (SPA) using React.

The main goal was to create not just a business card, but a digital experience that tells the brand's story, captures its unique, atmospheric character, and encourages customers to visit.

**Live Demo:** [https://czarnypuls.netlify.app/](https://czarnypuls.netlify.app/)
